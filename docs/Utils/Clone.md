## Clone (Utility)

## Usage (Basic)

```js
import { Clone } from 'react-native-mosaic';

const { data } = this.state;
let clonedData = Clone.deepClone(data);
```

## Instance Methods

| Method      | Params | Description              |
| ----------- | ------ | ------------------------ |
| `deepClone` | object | Deep clone a JSON object |
