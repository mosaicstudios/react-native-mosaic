## Logo Banner

## Usage (Basic)

```js
import { LogoBanner } from 'react-native-mosaic';

render() {
    let image = require('../../assets/images/splash.png')
    return <LogoBanner image={image} />
  }

```

## Props Available

| Prop     | Type     | Description         |
| -------- | -------- | ------------------- |
| `style?` | `style`  | Base image style.   |
| `image`  | `string` | Source of the image |
